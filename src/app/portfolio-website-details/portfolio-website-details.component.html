About

The Portfolio Website is my personal website I created to display my resume and projects. The website is implemented with Angular 7 and Bootstrap 4.

Role: UX Designer, Developer

[ Personal ]

Design

Content Plan

I started the project by laying out the content I needed the website to display. The most important contents I wanted the website to display were my educational background and skillset as well as direct access to my paper resume. Next, my projects are to take focus.

Inspiration and Competitor Analysis

Having previously done a resume website before, I kept the old design in mind as reference to use features I liked and for areas of improvement. Keeping the introduction and skills areas clean and straightforward is a quality adopted from my previous resume website. For ideas on highlighting my projects, I took a look at a number of other portfolio websites in the industry. I came upon Jihoon Suh's website and took inspiration from the large thumbnails for his projects as well as how the project details are organized in their respective pages.

Mission Statement

My Portfolio Website should communicate my personality to the visitor as well as readily provide access to my resume and portfolio of projects.

Webflow Prototype

Using Webflow, I was able to prototype the website before diving in to code it. Using Webflow's interface, I roughly organized my content and planned the aesthetic of the website.

View prototype on Webflow.
The Problem

Due to my lack of work experience, my resume mainly consists of projects. My paper resume is not the best medium for displaying my skills applied through my projects.

Implementation

I chose to use Angular as an opportunity to learn something new while working on this project. I was already familiar with Bootstrap so I utilized it as well along with Sass. After a few Angular tutorials, I began implementation of the website. Following my Webflow prototype, I laid the homepage out section by section. I also made sure all content was mobile-friendly every step of the way.

View source code on GitHub.
My Solution

A portfolio website gives me the platform to display my work and simultaneously allows me to apply my skills in implementing the website.

Finishing Touches

As the website was being filled with my content, I also slowly began to optimize the images a bit more by using SVGs for the icons and using transparent background images for the project thumbnails.
